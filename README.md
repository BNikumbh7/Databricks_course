# 🚀 Databricks Course

> A comprehensive hands-on Databricks learning repository covering PySpark, SQL, Delta Lake, Unity Catalog, Data Modeling, ETL Pipelines, and Real-World Data Engineering Projects.

---

## 📖 About

This repository is a collection of **Databricks notebooks, datasets, assignments, and projects** created while learning and practicing Data Engineering concepts using Databricks.

It focuses on practical implementation with real-world examples rather than theory.

Topics include:

- Databricks
- Spark SQL
- Data Modeling
- ETL Pipelines
- Unity Catalog
- Medallion Architecture
- Slowly Changing Dimensions (SCD Type 1 & Type 2)
- Incremental Data Loading
- Data Quality Checks
- Performance Optimization

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/BNikumbh7/Databricks_course.git
```

## Open in Databricks

1. Clone the GitHub repository into Databricks Repos.
2. Open the notebooks.
3. Upload the sample datasets (if required).
4. Execute the notebooks module by module.
